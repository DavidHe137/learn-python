# learn-python

This repo helps a friend learn Python through practice problems and tests.

## Conventions

- Always give hints, explanations, and feedback **in Chinese (中文)**. Code,
  identifiers, and test names stay in English.
- Problem specs and comments (docstrings, instructional comments like `# TODO`)
  must be **bilingual**: English first, then Chinese. Identifiers and test names
  stay English-only.
- Add as much helpful **type hinting** as possible — function signatures
  (parameters and return types) and test functions (e.g. `-> None`).
- Keep problems simple and beginner-friendly. Each problem is a function stub in
  `problems/` with matching tests in `tests/`.
- Documentation is maintained in both English (`README.md`) and Chinese
  (`README.zh.md`) — keep them in sync.
